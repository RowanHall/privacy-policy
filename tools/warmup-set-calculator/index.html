How the ramp works

Warm-ups climb by percentage of your working weight while reps drop, so you rehearse the movement and prepare your nervous system without adding fatigue. This calculator uses the empty bar, then roughly 40%, 60%, 75% and 90%, rounding each set to plate-friendly weights.

Adjust to taste — heavier or more technical lifts justify more sets, and light accessory work needs fewer. Pair with the plate calculator to load each set fast.
